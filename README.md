```md
# Student Performance Predictor

A predictive analysis project using Machine Learning.

## Features
- Predicts writing score
- Uses Random Forest ML model
- Flask backend
- HTML/CSS/JS frontend
- Interactive UI

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript

## How To Run

Install packages:

pip install -r requirements.txt

Run backend:

python app.py

Open index.html in browser.
```
